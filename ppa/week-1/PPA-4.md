## Question

Accept two integers as input and print their sum as output.



## Hint

We can accept multiple inputs on the same line:

```python
x, y = int(input()), int(input())
```

## Solutions


::: {.panel-tabset}
## Solution-1

```python
x = int(input())
y = int(input())
print(x + y)
```



## Solution-2

```python
x, y = int(input()), int(input())
print(x + y)
```
